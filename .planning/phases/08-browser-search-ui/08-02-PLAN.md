<objective>
Implement the entire client-side SPA controller in `static/js/app.js` using vanilla browser APIs only — no npm, no CDN, no framework. Purpose: Turn the static skeleton from Plan 01 into a working search interface. All state is module-level variables; all DOM nodes are cached at load; the three UI states are switched by setting `document.body.dataset.state`.

Output: New `static/js/app.js`.

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| ES highlight excerpt → DOM (innerHTML) | ES-controlled `<b>` fragments from the local index; trusted source (D-10) |
| scraped article body → DOM (textContent) | Wikipedia/blog text may contain stray HTML; treated as untrusted |
| browser → /api/search, /api/status | Query/filter params already sanitized server-side (T-07-01/02) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-04 | Tampering (XSS) | result excerpt rendering | mitigate | innerHTML used ONLY for `result.excerpt` (ES-controlled local-index highlight tags, D-10); verify gate caps innerHTML occurrences at 2 and asserts title/domain use textContent |
| T-08-05 | Tampering (XSS) | article body rendering | mitigate | `articleBody.textContent = result.body` — never innerHTML; scraped content cannot inject HTML/script (Pitfall 3); verify gate asserts articleBody.textContent |
| T-08-06 | Tampering (XSS) | result title / source_domain | mitigate | Assigned via textContent only; browser escapes all markup automatically (Don't-Hand-Roll: textContent escaping) |
| T-08-07 | Information Disclosure | stale-result race | mitigate | AbortController cancels the in-flight request before each new search; prevents a slow earlier response overwriting newer results (Pitfall 4) |
| T-08-SC | Tampering | npm/pip/cargo installs | accept | No package installs — vanilla browser APIs only; RESEARCH.md Package Legitimacy Audit confirms zero new packages |
</threat_model>
