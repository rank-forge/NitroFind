## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-12-04 | Injection (XSS) | pagination rendering in app.js | mitigate | `renderPagination` only sets boolean `disabled` flags and `renderResultCount` sets `statsLine.textContent` (not innerHTML) with a numeric total and time — no untrusted string is written as HTML by the pagination code |
| T-12-05 | Tampering | client `currentPage` state | accept | `currentPage` is client-only UI state; the server independently clamps `page` via `_safe_int_param` (12-02), so a manipulated client value cannot produce an invalid ES query |
| T-12-SC | Tampering | package installs | accept | No packages installed; edits to existing HTML/CSS/JS only — no supply-chain surface |
</threat_model>
