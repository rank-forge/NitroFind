<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| OS env (`PORT`) → app.run | Untrusted env value parsed into an int for the listen port |
| network interface → Flask | Bind address determines who can reach the server |
| SIGINT (Ctrl+C) → process | Operator signal triggers the shutdown path |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-01 | Information Disclosure | `app.run` bind address | mitigate | `host="127.0.0.1"` — never `0.0.0.0` (WSL-only, no external access) |
| T-06-02 | Elevation of Privilege | Werkzeug debug/reloader | mitigate | `debug=False, use_reloader=False` — also prevents the reloader forking a duplicate ES JVM (Pitfall 1) |
| T-06-03 | Tampering | `PORT` env var injection | mitigate | `int(os.environ.get("PORT", 5000))` wrapped in try/except ValueError → `sys.exit(1)` with a clear message |
| T-06-08 | Denial of Service | orphaned ES JVM after crash | mitigate | `try/finally` calls `shutdown_es(state["process"])` (None-guarded) on KeyboardInterrupt; verified by manual `ps aux` check |
</threat_model>
