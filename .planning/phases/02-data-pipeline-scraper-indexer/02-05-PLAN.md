Compose the four Wave-1 building blocks (cleaner, state, indexer, wikipedia, blogs) into a runnable CLI: `python scripts/scraper.py [--wikipedia|--blogs|--all]`. The live run is gated by a human-verify checkpoint because it requires internet access, a running ES node, and ≥10 minutes of scraping — Claude cannot reasonably automate the full live run inside the plan.
